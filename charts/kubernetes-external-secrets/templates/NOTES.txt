Supported backends in this build: AWS Secrets Manager (secretsManager) and AWS SSM Parameter Store (systemManager).

SECURITY: ExternalSecret `template` content is compiled and executed with lodash templating inside the controller process. Creating or editing an ExternalSecret template is therefore equivalent to code execution in the controller — restrict who may create ExternalSecrets via RBAC.
